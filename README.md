## 应用简介:
无疾 —— 即没有疾病。旨在提供疾病自查及医疗资讯的资讯类应用。

## 功能介绍:
1. 疾病自查: 
    可以按照身体部位和科室分类来查询疾病，也可以点击右上角搜索按钮按照疾病名称和症状查询疾病，点击结果可以浏览疾病简介、病因、预防、病状、药品、食品、检查等
2. 医学百科: 
    了解医学相关的资讯和知识，按照诸多标签进行分类，以新闻的形式呈现
3. 附近医院 : 
    以我的位置为中心，搜索方圆5公里内的医院，地图和定位由高德地图提供
